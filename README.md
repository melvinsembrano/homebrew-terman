# homebrew-terman

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://github.com/melvinsembrano/terman/blob/main/LICENSE)
[![Version](https://img.shields.io/badge/version-0.6.3-blue.svg)](https://github.com/melvinsembrano/terman/releases)

Homebrew tap for [terman](https://github.com/melvinsembrano/terman) — a Terminal API Client: TUI and CLI for building and running HTTP requests.

## Installation

```sh
brew tap melvinsembrano/terman
brew trust melvinsembrano/terman
brew install terman
```

## Supported Platforms

| OS    | Architecture        |
|-------|---------------------|
| macOS | Intel (x86_64)      |
| macOS | Apple Silicon (ARM64) |
| Linux | x86_64              |
| Linux | ARM64               |

## Usage

After installation, run:

```sh
terman
```

For full documentation, commands, and configuration options, visit the [main repository](https://github.com/melvinsembrano/terman).
